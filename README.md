## 一、简介
- 初学JAVA制作的坦克大战小游戏，使用JAVA 的GUI模拟2,5D界面

## 二、操作说明
- W,S,A,D键控制坦克的方向
- 鼠标控制射击方向，左键发射普通炮弹，右键发射穿透导弹
- 按下B键后引导一秒后可以回城

## 三、环境要求
- Eclipse
- JDK7或更高版本

## 四、部署说明
1. 在eclipse中导入java 项目。点击eclipse菜单File - Import，选择General - Existing Projects into Workspace。
2. 设置项目编码为utf-8，选择jdk1.8版本或以上。
3. 编译后，运行TankClient主类
